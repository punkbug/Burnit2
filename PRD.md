# 프로젝트명: 소각장 (Burn It) - 프라이버시 중심 감정 쓰레기통 앱

## 1. 프로젝트 개요
사용자가 거친 감정을 가감 없이 쏟아내고 불태워버리는 모바일 네이티브 앱(MVP)입니다. 다크하고 무거운 분위기에서 벗어나, 사용자가 앱을 켜는 순간부터 편안함을 느낄 수 있는 **온화하고 미니멀한 화이트 톤(Calm & Minimal)**으로 리뉴얼되었습니다.

## 2. 데이터 및 API 아키텍처
- **AI 연동 (Supabase Edge Function)**: 앱은 `supabase.functions.invoke('chat-gemini', ...)`를 호출해 위로 메시지를 받아옵니다. Edge Function 내부에서 Gemini 등 외부 LLM 호출을 처리합니다.
- **보안 (Key 관리)**: 앱에는 Gemini 키가 없으며, Supabase 접속 정보(`SUPABASE_URL`, `SUPABASE_ANON_KEY`)만 `flutter_dotenv`로 로드합니다.

## 3. 핵심 화면 및 기능 흐름 (Flow)
1. **Welcome Screen (첫 화면)**
   - "오늘 하루도 고생 많았어요" 같은 따뜻한 위로 문구와 편안한 일러스트(또는 아이콘) 배치.
   - 중앙 하단에 둥글고 부드러운 "감정 소각하러 가기" 버튼 위치.
   - **우측 상단에 감정 영수증 보관함 버튼을 배치해 과거 기록으로 바로 이동 가능.**

2. **Burn Screen (감정 쏟아내기)**
   - 텍스트 입력창은 그림자가 살짝 들어간 깔끔한 흰색 카드 형태.
   - **입력창 바로 아래에** AI 페르소나 선택 버튼(🔥같이 화내기, 💧현실적 조언, 🪵따뜻한 위로)을 가로로(Row) 배치하여 하나를 선택(Toggle)할 수 있도록 UI 구성.
   - '소각하기' 버튼 클릭 시, "AI가 당신의 감정을 읽는 중..."이라는 로딩 인디케이터가 잠시 표시된 후 시각적 카타르시스 애니메이션과 햅틱 진동 실행.

3. **Catharsis & Feedback (소각 및 위로 팝업)**
   - 텍스트가 타들어 가며 사라진 직후, 선택한 페르소나의 실제 Gemini AI 위로 메시지가 부드러운 팝업(Overlay)으로 등장.
   - 팝업은 **화면 정중앙**에 표시되며, 사용자가 **「닫기」**를 눌렀을 때만 닫힌다.
   - 닫기 후 입력창의 모든 텍스트 상태가 초기화(Clear)되며 다시 첫 화면(Welcome Screen)으로 복귀한다.

4. **Receipt Screen (감정 영수증 보관함)**
   - 로컬(기기)에만 저장된 과거 기록(날짜, 감정 온도, AI 메시지)을 리스트로 확인.
   - **개별 삭제 기능**: 사용자가 원할 경우 스와이프나 휴지통 버튼으로 특정 영수증을 영구 삭제 가능.
   - **전체 삭제(초기화) 기능 제공**: 보관함 내 모든 기록을 한 번에 비울 수 있는 기능 제공.
